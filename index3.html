Image Steganography Using LSB Substitution

Hide and retrieve secret messages within images using Python and OpenCV.

Project Overview

Goal: To implement an image steganography technique using Least Significant Bit (LSB) substitution. This method embeds secret information into the least significant bits of pixel values, making it nearly undetectable to the naked eye.

Objectives

  • Select a cover image where secret data will be hidden.
  • Implement LSB substitution for embedding secret messages within images.
  • Extract hidden data from stego-images using reverse LSB substitution.
  • Ensure minimal distortion and secure hiding of messages.

Data Acquisition and Preprocessing

Cover Image

Choose a high-quality image that can hide data effectively. Ensure that the resolution is sufficient to store the desired message length.

Preprocessing

  • Image format: Use lossless image formats like PNG or BMP for reliable data embedding.
  • Image resizing: Resize images for uniformity and ease of processing.

LSB Substitution Algorithm

Embedding Process

  • Convert the secret message into a binary format.
  • Modify the least significant bit of each pixel in the cover image to store the message.
  • Ensure that the modified image (stego-image) looks visually similar to the original.

Extraction Process

  • Read the least significant bits of the stego-image.
  • Reconstruct the binary message and convert it back to its original form.

Evaluation

Quality Metrics

  • PSNR (Peak Signal-to-Noise Ratio): Measure the similarity between the cover image and the stego-image.
  • MSE (Mean Squared Error): Calculate the average squared difference between the pixel values of the original and stego-images.

Performance Testing

Test the model with varying message lengths and image sizes to ensure data integrity and minimal visual distortion.

Deployment

Web Application

Develop a web interface using Python (Flask/Streamlit) to allow users to upload images and hide/retrieve messages.

  • Allow users to upload cover images for embedding messages.
  • Provide an interface for users to extract hidden messages from stego-images.

Cloud Deployment

Deploy the application on a cloud platform (e.g., Streamlit Cloud, Heroku) for easy access.
